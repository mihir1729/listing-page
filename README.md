# Github User Listed Repositories

## Setting Up Locally

### 1. Clone Repository
- Save the code to your local machine

### 2. Installing Dependencies
- `npm install`

### 3. Development Environment
- `npm start`

### 4. Open Browser
- `http://localhost:3000/`


## Active Deployment

### - The project is hosted live on netlify on the below url.
